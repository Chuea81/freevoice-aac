# Google Play Store Listing

This folder contains all metadata and copy needed for Google Play Store submission of FreeVoice AAC.

## Files

- **SHORT_DESCRIPTION.txt** — 80-character app tagline
- **FULL_DESCRIPTION.txt** — 4000-character app description (main selling points)
- **KEYWORDS.txt** — 10 search keywords for discoverability
- **SUBMISSION_CHECKLIST.md** — Pre-submission quality checklist
- **privacy-policy.txt** — Privacy policy (if not already on website)
- **terms-of-service.txt** — Terms (if needed)

## Next Steps

1. **Build Release APK** — `npm run build:apk:release`
2. **Test on Real Devices** — Verify all features work
3. **Review Checklist** — Run through SUBMISSION_CHECKLIST.md
4. **Prepare Screenshots** — Create 5–8 showcase screenshots
5. **Create Google Play Account** — Pay $25 developer fee
6. **Submit to Play Store** — Upload APK and metadata in Play Console

## Key Features to Highlight

- **Completely free** (no subscriptions, no ads)
- **1,100+ symbols** across 66 boards
- **10 languages** with professional translations
- **Offline-capable** (works without WiFi)
- **Customizable** (add photos, create boards)
- **Private** (no tracking, no data collection)
- **Open-source** (code available on GitHub)

## Target Audience
